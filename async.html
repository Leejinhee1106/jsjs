async

- 다른 함수와 병렬로 실행되는 함수를 비동기라고 함.
- Ex: setTimeout()
- 병렬: 여러가지 일들이 동시에 이루어짐 / 여러 요소가 나란히 놓여 있음
- 컴퓨터에서 병렬 뜻: 여러 작업 처리를 동시에 수행하는 방식
- JS에서는 비동기적으로 작업을 처리할 수 있어서 병렬로 진행되는 것 처럼 보임

Promise

- 결과를 약속 드립니다.

0. Promise Object 속성
- Pending(보류) : 정의되지 않음(undifinded)
- fullfilled(성취) : 결과값(a result Value)
- rejected(거부) : 오류객체

1. 코드생성 [producing code] : 시간이 걸릴 수 있는 코드
생성코드가 결과를 얻으려면 아래에 두 콜백중에 하나를 호출해야 함
Success : myResolve(result value)
Error : myReject(error object)

2. 소비코드 [Consuming code] : 결과를 기다려야 하는 코드
자바스크립트, 자바, 리액트, 뷰는 객체지향언어
생성코드와 소비코드를 연결하는 객체를 Promise
